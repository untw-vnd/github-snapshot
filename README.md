# github-snapshot

Export GitHub entities to email-able files.

Paste a project URL, get a spreadsheet with one row per issue and one column per custom field. 
Paste an issue URL, get a PDF with the issue body, all comments, and a metadata header.

## Try it

An unauthenticated demo is available at https://github-snapshot.tailstory.workers.dev. Paste any public GitHub project or issue URL.

## Deploy your own

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for the fork-and-deploy walkthrough. Each deployment is independent — credentials and access live on your own Cloudflare account.
